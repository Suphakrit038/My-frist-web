ประวัติความเป็นมาของจังหวัดสุราษฎร์ธานี

สุราษฎร์ธานีเป็นจังหวัดในภาคใต้ของประเทศไทยที่มีประวัติศาสตร์และวัฒนธรรมอันยาวนาน โดยพื้นที่นี้มีการตั้งถิ่นฐานของมนุษย์มาตั้งแต่ยุคก่อนประวัติศาสตร์ ในยุคอาณาจักรศรีวิชัย สุราษฎร์ธานีเป็นศูนย์กลางการค้าและการเผยแพร่พระพุทธศาสนา
ในสมัยกรุงศรีอยุธยาและกรุงธนบุรี สุราษฎร์ธานีเป็นเมืองท่าที่สำคัญทางการค้า และในสมัยรัตนโกสินทร์ รัชกาลที่ 5 ได้เปลี่ยนชื่อเมืองจาก "เมืองไชยา" เป็น "สุราษฎร์ธานี" ซึ่งหมายถึง "เมืองแห่งคนดี"
ปัจจุบัน สุราษฎร์ธานีมีเศรษฐกิจที่เจริญรุ่งเรือง โดยเฉพาะในด้านการเกษตร การประมง และการท่องเที่ยว เช่น เกาะสมุย เกาะพะงัน และเขาสก
